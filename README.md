## Praktikum 1 – Class dan Object
Pada praktikum ini saya mempelajari cara membuat class dan object dalam Python.
Class Hero dibuat sebagai cetakan, sedangkan object hero dibuat dari class tersebut.

**Konsep OOP:** Class & Object  
**Hasil:** Program berhasil menampilkan data hero.

## Praktikum 2 – Interaksi Antar Object
Pada praktikum ini, object dapat saling berinteraksi menggunakan method serang dan diserang.

**Konsep OOP:** Method & Object Interaction  
**Hasil:** Hero dapat saling menyerang dan HP berkurang sesuai damage.

---

## Praktikum 3 – Inheritance
Pada praktikum ini diterapkan konsep pewarisan dengan membuat class Mage yang mewarisi class Hero menggunakan fungsi super().

**Konsep OOP:** Inheritance  
**Hasil:** Mage memiliki kemampuan tambahan berupa skill khusus.

---

## Praktikum 4 – Encapsulation
Pada praktikum ini atribut HP diamankan menggunakan private attribute dan diakses melalui getter dan setter.

**Konsep OOP:** Encapsulation  
**Hasil:** HP tidak bisa diubah sembarangan dan tervalidasi.

---

## Praktikum 5 – Abstraction & Interface
Pada praktikum ini dibuat abstract class GameUnit yang memaksa class turunan memiliki method serang dan info.

**Konsep OOP:** Abstraction  
**Hasil:** Program mencegah class yang tidak lengkap.

---

## Praktikum 6 – Polymorphism
Pada praktikum ini satu method serang menghasilkan perilaku berbeda pada tiap class hero.

**Konsep OOP:** Polymorphism  
**Hasil:** Loop yang sama menghasilkan output berbeda.
